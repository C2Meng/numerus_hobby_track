# numerus_hobby_track

## Problem
Have many hobbies, no time to do all.

---

## Tech Stack

### Frontend
- Django Templates
- HTML
- CSS
- JavaScript

### Backend
- Python
- Django
- SQLite

### Version Control
- Git
- GitHub

---

## Core Features

1. Create, update, or delete hobbies  
2. List hobbies by priority  
3. Allocate time to hobbies  
   - Possible integration with calendar apps (Google Calendar, Microsoft Teams)  
4. View schedule
